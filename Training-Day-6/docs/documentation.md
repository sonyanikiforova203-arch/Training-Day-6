echo -  встроенная команда, одна из самых часто исписываемых команд  Bash-скриптах. она выводит обычный текст или значения переменных в терминал.  также она может оформлять сообщения и выводить результаты выполнения команд. применяется, когда нужно вывести информацию пользователю, отобразить ошибку, вывести результаты работы скрипта и оформить меню и отчёты

read - встроенная команда Bash, которая ожидает ввод пользователя с клавиатуры и сохраняет его в переменную. основные возможности - это получить данные от пользователя, сохраненные введённого текста, создание интерактивных программ. применяется, когда нужно ввести имя пользователя, запросить название проекта, подтвердить действия, настроить параметры программы 

if - это оператор bash, который позволяет выполнять разные действия в зависимости от результата проверки. если одни условия выполняются один блок команд, если нет второй блок. основные возможности этой команды - проверять существование файлов, существование папок, успешность выполнения предыдущих команд, любые логические условия. применяется, когда нужно проверить наличие файлов, проверить ошибки, принять решение внутри программы и что не мало важно создание более умных Bash-скриптов

for - цикл, который позволяет выполнять одну и ту же группу команд несколько раз. будет полезен, когда нужно перебрать файлы, выполнить одну команду много раз, автоматически обработать несколько объектов. применяется, когда нужно обработать больше количество файлов, выполнить резервное копирование, автоматически установить программу, запустить несколько экспериментов в AI-проектах

while - цикл, который выполняет команды до тех пор, пока условие остаётся актуальным. в отличие от for, количество повторений заранее неизвестно. эта команда может повторять действия, ожидать выполнения условий, работать с пользовательским вводом, сдавать бесконечные циклы. применяется в ожидание запуска программы, чтение данных, опрос пользователя, автоматический мониторинг системы

chmod +x - команда Linux для изменения прав доступа к файлам. параметр +x добавляет право на выполнение файла как программы.
chmod - изменить права
+ - добавить право
x - разрешить выполнение
применяется, когда нужно запустить Bash-скрипт, запустить Python-скрипт. выполнить собственную программу и подготовить рабочие скрипты для серверов и проектов


Project Setup Script - скрипт подготовки нового проекта. это полезно для того, чтобы в момент создания нового проекта разработчику не вводить несколько команд подряд, а начать проект с одинаковой структуры. в этом скрипте можно задать команды для создания папок и файлов. актуальными в этом случае будут команды mkdir, touch, echo, chmod, cp, mv
